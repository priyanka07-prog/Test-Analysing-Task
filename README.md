# 📈 Bitcoin Market Sentiment vs Trader Performance Analysis

## Primetrade.ai Data Science Internship Assignment

An end-to-end data analysis project that explores the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

---

# 📌 Project Objective

The objective of this project is to investigate how Bitcoin market sentiment influences trader behavior and profitability.

By combining historical trading records with the Fear & Greed Index, this project aims to:

- Analyze trader performance under different market conditions
- Identify profitable and loss-making trading patterns
- Perform statistical analysis to validate findings
- Develop trader personas
- Recommend actionable trading strategies based on data

---

# 📊 Datasets Used

## 1. Hyperliquid Historical Trading Data

Contains historical trading records including:

- Account
- Coin
- Execution Price
- Size (USD)
- Side (Buy/Sell)
- Timestamp
- Closed PnL
- Fee
- Transaction Details

---

## 2. Bitcoin Fear & Greed Index

Contains daily Bitcoin market sentiment:

- Date
- Classification (Fear / Greed)

# 📌 Key Insights

- Market sentiment influences trader activity and profitability.
- Profitable traders generally maintain higher win rates than loss-making traders.
- Trading risk varies significantly across trader segments.
- Statistical testing was used to validate differences in profitability under Fear and Greed market conditions.
- Data-driven trading strategies can improve decision-making and reduce trading risk.
